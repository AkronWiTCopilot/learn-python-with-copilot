# Learn Python with GitHub Copilot 

## Conclusion

Using Copilot, Jane was able to learn Python to create a project that manages tasks. She learned how to create a project class, task class, and how to add tasks to a project. She also learned how to document her project code using docstrings, inline comments, example code, TODO comments, and evaluate different solutions using code alternatives. Finally, she learned how to test her project in various scenarios using tools such as Pytest, unittest, and MagicMock.
